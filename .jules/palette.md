## Critical Learnings
- This project is a Python wrapper for the GHC compiler. It distributes a Haskell compilation toolchain (GHC) alongside its build system (Cabal) using standard Python packaging mechanisms (PEP 427 wheels).
- There is NO UI in this project (no frontend, no desktop app, no CLI beyond standard python scripts).
- Therefore, there is no UX work to be done.
